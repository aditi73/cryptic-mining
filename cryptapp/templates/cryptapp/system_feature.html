From cipher-text to plain-text..

A System to convert cipher-text to plain-text using English grammar rules.

This system is implemented for substitution cipher as well as transposition cipher.

The algorithm uses various knowledge sources(i.e English dictionary words) that are applied to the input cipher-text so as to obtain corresponding plain-text.

Let us assume that the user has entered following cryptogram: “q azws dssc kas dxznn dasnn”.
